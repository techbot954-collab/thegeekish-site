# Geekish Static Image Posts

These SVGs are 1080x1350 feed images matched to the current Geekish articles.

Reference direction from the user video:
- Discover/news-feed feel
- big visual area first
- bold headline below
- source/author/time style line
- quick CTA to read the article

Use these as fast image posts. They can be exported to PNG in Canva/Figma, browser screenshot, or any SVG-to-PNG tool.

Preview all image posts by opening `social-images/index.html`.
